📊 Sales Data Analysis

This project focuses on analyzing sales data to uncover key insights related to customer behavior, revenue trends, product performance, and seasonal variations. It includes end-to-end data analysis—covering data cleaning, transformation, SQL analysis, and visualizations.

📁 Project Overview

The goal of this project is to:

Understand customer purchasing patterns
Analyze product-level performance
Identify high-revenue categories
Study seasonal sales trends
Generate actionable insights for business decision-making

This project is useful for anyone exploring data analytics, EDA, SQL, or business intelligence.

📂 Dataset

The dataset contains information such as:

Customer ID
Gender
Age
Item Purchased
Category
Purchase Amount (USD)
Season
Review Rating
Previous Purchases
Discount Applied
Payment Method
